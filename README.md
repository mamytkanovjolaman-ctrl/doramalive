# DoramaLive

Азиатские сериалы онлайн.

## Как запустить локально

```bash
npm install
npm start
```

## Как выложить на Vercel

1. Загрузи эту папку на GitHub
2. Зайди на vercel.com
3. Нажми "Add New Project"
4. Выбери репозиторий
5. Нажми Deploy — готово!

## Следующие шаги

- Подключить Kodik API (kodik.info)
- Подключить Supabase для авторизации
- Зарегистрировать домен
